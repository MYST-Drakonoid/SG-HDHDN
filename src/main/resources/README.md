# Handheld DHD for SGJourney

## Description
This mod is an addon for SGJourney (Minecraft 1.21.1, NeoForge 21.1.209).
It adds:
- Address Database Editing Blocks
- Handheld DHD
- Iris Control & security Blocks
- Global database integration for Stargate network dialling
